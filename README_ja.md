# FairSoil

FairSoil は、誠実さと正直さが短期的な搾取やコストの外部化よりも報われる「公正な土壌」を育むことを目的とした、分散型ベーシックインカム（UBI）プロトコルです。

## ビジョン
以下を実現するデジタル経済基盤をつくること:
- 「誠実」で「信頼できる」ことが経済的に有利であること。
- UBI によって、誰もが望ましくない仕事を「断る自由」を持てること。
- 社会的に必要だが困難・危険なタスクが、動的なインセンティブによって公正に報われること。

## コアメカニズム

### 1. デュアルトークンシステム
- **トークン A（Flow）:** 日常取引のための「減価」する通貨。
  - 認証済みのすべての人に毎日配布。
  - 時間とともに自動的に価値/量が減少（例: 30 日の半減期）し、流通を促進。
- **トークン B（Asset）:** 永続的な価値の保管手段。
  - 高い誠実性の行動や社会的に不可欠な貢献を通じて獲得。
  - トークン A の単純な貯蔵では獲得できない。

### 2. 誠実性に基づく評価
- 報酬は「誠実性」（単に「良いことをする」ではなく「不正をしない」）に基づく。
- 「コストの外部化」（リスク/コストを他者に押し付けること）を罰する。
- 安全性および手順基準が満たされたことを検証するために、プロセスの証明を使用。

### 3. Sybil 耐性
- **World ID** と統合し、各参加者が一意の人間であることを保証。

### 4. 社会的に不可欠なタスクへの動的インセンティブ
- 供給が不足している緊急・高リスク・必須タスク（例: 災害救援、医療）に対して、報酬を自動的にスケール。

## 設計メモ（議論のたたき台）

### 1. 「逃げ得」「言い逃れ」を技術的に塞ぐ
- **先行預託（Collateral First）:** 事後の罰金ではなく、活動開始前に補償用デポジットをロック。
- **署名付き指示（Cryptographic Instruction）:** 重要指示は署名必須。後から責任の所在が辿れる。
- **金銭以外のペナルティ:** SBT を World ID に紐付け、重大な背任は参加制限などの社会的コストへ。

### 2. 「声の大きさ」や消耗戦に勝たせない
- **ブラインド裁定:** 当事者情報を隠し、ログと署名のみで判定。
- **シェリングポイント型の報酬:** 多数一致の判定が得になる設計で、事実へ収束。
- **反消耗戦:** ルール逸脱が検知された時点で資金をロックし、長期化の不利を低減。
- **Skin-in-the-Log:** 口約束ではなく、実行ログと署名完了のみを信頼指標に。

### 3. ぎちぎちにしすぎないための「余白」
- **UBIは無条件:** 生活基盤は評価に連動させない。
- **オン/オフの切り替え:** 高リスク領域のみ厳格プロトコル、私的領域は自由。
- **忘却と再起:** スコアは時間で減衰し、過去の失敗が永続的に縛らない。
- **「誠実さ」は保険:** まっとうなログが、いわれのない攻撃から守る盾になる。

### 4. 記録は「全歩行」ではなく「責任移転の節目」
- **Gitのコミット方式:** 重要なマイルストーンのみを記録し、過程は裁量に任せる。
- **相互署名と抜き打ち:** 納品・検品などの節目は多者署名で証跡を連鎖。
- **事後検証の許容:** 常時監視ではなく、問題時にログを掘り返せる設計。
- **仕様バグの防止:** 手順が不可能な場合は「実行不能フラグ」で異議申立を可能に。

### 5. 歴史/経済のエピソードからの学び
- **海賊船の乗船前合意:** 仕事ごとに契約テンプレを合意し、署名後に開始。
- **無人店舗のライブ配信:** 常時監視ではなく「必要時に証明できる」状態で自律を促す。
- **バンコール的均衡:** 収益が偏りすぎた場合、土壌（UBI）へ緩やかに還元。

### 6. 「挑戦者が萎える」副作用への対処
- **保有量ではなく停滞に課税:** 動いている資産は減らない設計。
- **成功の猶予期間:** 失敗リスクを取った成果に一定の減価免除期間を付与。
- **寄与を名誉へ変換:** 還元は没収ではなく社会的評価（投票権など）へ。
- **先駆者報酬:** 先に道を切り開いた人へ、後発の成功から還元する仕組み。

## 技術スタック（予定）
- **スマートコントラクト:** Solidity
- **開発フレームワーク:** Foundry または Hardhat
- **フロントエンド:** Next.js（React）
- **ブロックチェーン:** Ethereum L2（Optimism / Polygon）
- **アイデンティティ:** World ID SDK
